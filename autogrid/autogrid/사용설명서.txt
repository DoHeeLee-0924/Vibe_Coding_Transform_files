  AutoGrid - AIMZ 얼굴 격자 오버레이 도구
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━


■ 설치 방법
──────────────────────────────────────────

  1. 압축 파일을 원하는 위치에 풀어주세요.
  2. 폴더 안의 index.html 파일을 더블클릭하여 실행합니다.
     (Chrome 또는 Edge 브라우저 권장)

  ※ 별도 설치 프로그램이 필요 없습니다.
  ※ 인터넷 연결이 필요합니다 (얼굴 감지 AI 모델 로딩).


■ 기본 사용법
──────────────────────────────────────────

  1. 이미지 불러오기
     - [이미지 업로드] 버튼 클릭
     - 캔버스 영역에 파일 드래그 앤 드롭
     - Ctrl+V 로 클립보드 이미지 붙여넣기

  2. 격자 적용
     - [얼굴 자동 감지] : AI가 얼굴을 찾아 해당 영역에만 격자 적용
     - [전체 격자]      : 이미지 전체에 격자 적용

  3. 결과 저장
     - [저장] 버튼 클릭 시 PNG 파일로 다운로드
     - [저장 폴더 지정] 으로 폴더를 선택하면
       이후 저장 시 해당 폴더에 자동 저장됩니다.


■ 격자 설정 (좌측 사이드바)
──────────────────────────────────────────

  간격    격자 선 사이의 간격 (px)
  굵기    격자 선의 두께 (px)
  투명도  격자 선의 투명도 (0~100%)
  여백    얼굴 영역 주변 추가 여백 (%)
  선 색상 격자 선 색상 선택


■ 수동 편집 (얼굴 감지 보정)
──────────────────────────────────────────

  자동 감지가 정확하지 않을 때 수동으로 보정할 수 있습니다.

  [+ 추가]  : 활성화 후 캔버스를 드래그하여 얼굴 영역 직접 지정
  [- 제거]  : 활성화 후 잘못된 영역을 클릭하여 제거
  [격자 적용] : 수정된 영역 목록에 격자를 최종 적용


■ 권장 환경
──────────────────────────────────────────

  브라우저  Google Chrome / Microsoft Edge (최신 버전)
  OS       Windows 10 이상 / macOS 12 이상
